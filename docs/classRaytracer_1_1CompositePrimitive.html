Detailed Description

Implements the Composite pattern for 3D primitives.

This class allows grouping multiple primitives and treating them as a single entity. It implements the Composite design pattern, enabling hierarchical organization of scene objects. When a ray intersects with a composite, it tests intersection with all contained primitives and returns the closest hit.
