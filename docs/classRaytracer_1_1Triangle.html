Detailed Description

Represents a triangle primitive for raytracing.

The Triangle class defines a triangle in 3D space, characterized by three vertices. It provides methods for ray-triangle intersection testing using the Möller–Trumbore algorithm and normal vector calculation which are essential for the raytracing process.

Tests if a ray intersects with this triangle.

Calculates if and where a ray intersects with this triangle using the Möller–Trumbore algorithm. The intersection distance is stored in the parameter t if an intersection is found.
