Detailed Description

Interface for all geometric primitives.

This abstract class defines the interface that all renderable primitives (spheres, planes, cylinders, etc.) must implement to be compatible with the raytracing engine. It provides pure virtual methods for ray intersection, normal calculation, and basic property access.
