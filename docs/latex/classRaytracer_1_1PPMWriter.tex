\doxysubsection{Detailed Description}
Handles writing of images to PPM format files. 

Provides functionality to save rendered images from the ray tracer to PPM files. Can operate either with a \doxylink{classRaytracer_1_1Renderer}{Renderer} reference or directly with an image buffer.
