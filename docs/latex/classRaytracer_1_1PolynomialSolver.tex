\doxysubsection{Detailed Description}
Static class for solving polynomial equations up to degree 4. 

Provides numerical methods for finding real roots of\+:
\begin{DoxyItemize}
\item Quadratic equations (degree 2)
\item Cubic equations (degree 3)
\item Quartic equations (degree 4)
\end{DoxyItemize}

Used primarily for computing ray-\/object intersection points.
