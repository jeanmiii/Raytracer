\doxysubsection{Detailed Description}
3D vector with common mathematical operations 

Represents a vector in 3D space with x, y, z components. Provides essential operations for vector mathematics including\+:
\begin{DoxyItemize}
\item Arithmetic operations
\item Dot and cross products
\item Vector normalization
\item Length calculations 
\end{DoxyItemize}
