\doxysubsection{Detailed Description}
Represents a triangle primitive for raytracing. 

The \doxylink{classRaytracer_1_1Triangle}{Triangle} class defines a triangle in 3D space, characterized by three vertices. It provides methods for ray-\/triangle intersection testing using the Möller–\+Trumbore algorithm and normal vector calculation which are essential for the raytracing process.
