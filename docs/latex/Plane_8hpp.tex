Infinite plane primitive implementation for the raytracer.

\end{DoxyAuthor}
\begin{DoxyDate}{Date}

This file contains the Plane class which implements the IPrimitive interface. It represents an infinite plane in 3D space that can be rendered in the raytracing engine. A plane is defined by its normal vector and distance from the origin along that normal.
