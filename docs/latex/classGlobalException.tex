\doxysubsection{Detailed Description}
Custom exception class for the \doxylink{namespaceRaytracer}{Raytracer} project. 

This exception class extends std\+::exception to provide custom error handling with descriptive messages. It\textquotesingle{}s used throughout the project to handle various error conditions consistently.
