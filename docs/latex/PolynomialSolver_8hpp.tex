\doxysubsection{Detailed Description}
Polynomial equation solver for ray-\/object intersections. 

The Polynomial\+Solver class provides static methods for solving quadratic, cubic, and quartic polynomial equations. These are essential for calculating ray-\/object intersections in the ray tracer, particularly for complex shapes like toruses. The solver uses a combination of analytical methods and Newton-\/\+Raphson refinement.
