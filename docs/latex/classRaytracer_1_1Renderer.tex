\doxysubsection{Detailed Description}
Main rendering engine that converts a \doxylink{classRaytracer_1_1Scene}{Scene} into a 2D image. 

The \doxylink{classRaytracer_1_1Renderer}{Renderer} takes a \doxylink{classRaytracer_1_1Scene}{Scene} description and produces a 2D image by shooting rays through each pixel, computing intersections with scene objects, and applying lighting calculations. It supports\+:
\begin{DoxyItemize}
\item Primary ray casting
\item Recursive reflections
\item Blinn-\/\+Phong shading model
\item Hard shadows
\item \doxylink{classRaytracer_1_1Camera}{Camera} transformations 
\end{DoxyItemize}
