\doxysubsection{Detailed Description}
Factory class for creating geometric primitives in a 3D scene. 

Provides static methods to create various geometric shapes that implement the \doxylink{classRaytracer_1_1IPrimitive}{IPrimitive} interface. Implements the Factory design pattern to encapsulate object creation logic.
