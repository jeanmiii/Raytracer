\doxysubsection{Detailed Description}
Cone primitive implementation for the raytracer. 

\begin{DoxyAuthor}{Author}
EPITECH 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
2025
\end{DoxyDate}
This file contains the Cone class which implements the IPrimitive interface. It represents a cone in 3D space that can be rendered in the raytracing engine. The cone can be truncated to a finite height and can be rotated in 3D space.
