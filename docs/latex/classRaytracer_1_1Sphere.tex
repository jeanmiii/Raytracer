\doxysubsection{Detailed Description}
Represents a sphere primitive for raytracing. 

The \doxylink{classRaytracer_1_1Sphere}{Sphere} class defines a sphere in 3D space, characterized by its center point and radius. It provides methods for ray-\/sphere intersection testing and normal vector calculation which are essential for the raytracing process.
