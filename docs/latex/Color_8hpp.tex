\doxysubsection{Detailed Description}
RGB Color representation for the ray tracer. 

The Color class represents an RGB color with 8-\/bit precision per channel (0-\/255). It provides basic color manipulation capabilities including component access, modification, and value clamping to ensure valid color ranges.
