\doxysubsection{Detailed Description}
3D vector implementation for geometric calculations 

The Vector3 class represents a 3-\/dimensional vector with common operations needed for ray tracing and 3D graphics. It provides both component-\/wise access and mathematical operations including dot product, cross product, and normalization.
