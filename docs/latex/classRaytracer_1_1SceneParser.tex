\doxysubsection{Detailed Description}
Parser for scene configuration files. 

This class provides functionality to read and parse scene description files in libconfig format. It extracts scene elements such as camera properties, lights, and various geometric primitives, then populates a \doxylink{classRaytracer_1_1Scene}{Scene} object with these elements.
