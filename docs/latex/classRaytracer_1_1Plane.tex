\doxysubsection{Detailed Description}
Represents an infinite plane primitive for raytracing. 

The \doxylink{classRaytracer_1_1Plane}{Plane} class defines an infinite plane in 3D space, characterized by its normal vector and distance from the origin. It provides methods for ray-\/plane intersection testing and normal vector access which are essential for the raytracing process.
