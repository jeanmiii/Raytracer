Torus primitive implementation for the raytracer.

\begin{DoxyDate}{Date}

This file contains the Torus class which implements the IPrimitive interface. It represents a torus (donut shape) in 3D space that can be rendered in the raytracing engine. The torus is defined by its center, major radius, minor radius, and rotation.
