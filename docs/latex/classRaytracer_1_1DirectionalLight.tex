\doxysubsection{Detailed Description}
Implements directional lighting for a 3D scene. 

\doxylink{classRaytracer_1_1DirectionalLight}{Directional\+Light} represents a light source with parallel rays coming from a specific direction, simulating distant light sources like the sun. The light intensity is constant regardless of distance from the source.
