\doxysubsection{Detailed Description}
Implements ambient lighting for a 3D scene. 

\doxylink{classRaytracer_1_1AmbientLight}{Ambient\+Light} represents non-\/directional, uniform lighting that affects all objects in the scene equally, simulating indirect illumination. It inherits from \doxylink{classRaytracer_1_1ILight}{ILight} interface and provides basic ambient lighting functionality.
