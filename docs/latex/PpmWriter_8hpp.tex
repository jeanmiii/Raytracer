\doxysubsection{Detailed Description}
PPM image file writer for the ray tracer. 

The PPMWriter class handles the export of rendered images to PPM (Portable Pix\+Map) format. It supports both direct saving from a Renderer instance and static writing of image buffers. The PPM format is used for its simplicity and compatibility with many image viewers.
