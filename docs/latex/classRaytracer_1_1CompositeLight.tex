\doxysubsection{Detailed Description}
Implements the Composite pattern for light sources. 

This class allows grouping multiple light sources and treating them as a single entity. It\textquotesingle{}s an implementation of the Composite design pattern, which enables creating a tree structure of light objects in the scene.
