\doxysubsection{Detailed Description}
Represents a 3D scene containing camera, objects, and lights. 

The \doxylink{classRaytracer_1_1Scene}{Scene} class manages all elements of a 3D scene including the camera, geometric primitives (objects), light sources, and ambient lighting.
