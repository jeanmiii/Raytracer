\doxysubsection{Detailed Description}
Represents a cylinder primitive for raytracing. 

The \doxylink{classRaytracer_1_1Cylinder}{Cylinder} class defines a cylinder in 3D space, characterized by its base center, radius, height, and orientation. It provides methods for ray-\/cylinder intersection testing and normal vector calculation which are essential for the raytracing process.
